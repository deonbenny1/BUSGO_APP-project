BusGo - Premium Bus Ticket Booking App
🚀 Overview

BusGo is a modern Flutter-based bus ticket booking application inspired by premium travel apps like RedBus. The app provides a clean and interactive UI for searching buses, selecting seats, choosing boarding and dropping points, making payments, and managing bookings.

Built using Flutter and Dart, BusGo focuses on providing a smooth booking experience with premium UI/UX design.
